# DeenPal Blog — Topic Backlog

Curated, vetted, evergreen topics for the daily multilingual blog generator.
Each is a safe, well-documented subject with standard references (Quran + authentic
Sunnah). The daily routine should pick the **next 2 unused topics**, mark them
`[x]`, write content in all 4 languages (EN/NL/DE/FR), and run
`tools/deenpal_blog.py`.

**Rules for quality & SEO safety**
- Only use standard, non-controversial subjects. When a topic touches fiqh
  differences, present the mainstream view neutrally; do not issue fatwas.
- Every claim about a hadith must name a real collection (Bukhari, Muslim,
  Tirmidhi, Abu Dawud, An-Nasa'i, Ibn Majah). If unsure, omit it.
- Each post: unique angle, 700-1100 words, 1-2 internal links to existing posts
  **that exist in that language**, accurate Arabic + transliteration.
- Never reuse a slug already present in `blog/`, `nl/blog/`, `de/blog/`, `fr/blog/`.
